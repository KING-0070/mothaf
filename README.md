# متحف أرض اللبان | Museum of the Land of Frankincense

موقع تعريفي لمتحف أرض اللبان في منتزه البليد الأثري بصلالة — موقع التراث العالمي لليونسكو.

## التشغيل محلياً

افتح `index.html` في المتصفح، أو شغّل خادماً محلياً:

```bash
npx serve .
```

## الملفات

| ملف | الوصف |
|-----|--------|
| `index.html` | الصفحة الرئيسية (4 لغات) |
| `style.css` | التصميم والتجاوب |
| `script.js` | اللغات، القائمة، المعرض |
| `favicon.svg` | أيقونة الموقع |

## النشر

يتم النشر تلقائياً على GitHub Pages عند الدفع إلى فرع `main` (انظر `.github/workflows/static.yml`).

## المحتوى

- الصور من [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:Museum_of_the_Frankincense_Land)
- الإحداثيات والهاتف من مصادر عامة موثّقة
- يُنصح بالتحقق من ساعات العمل والأسعار من المصادر الرسمية قبل الزيارة

## الترخيص

المحتوى التعليمي مفتوح للاستخدام. صور ويكيميديا تخضع لتراخيصها الخاصة.
