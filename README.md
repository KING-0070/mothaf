# متحف أرض اللبان | Museum of the Land of Frankincense

موقع تعريفي لمتحف أرض اللبان في منتزه البليد الأثري بصلالة — موقع التراث العالمي لليونسكو.

## التشغيل محلياً

افتح `index.html` في المتصفح، أو شغّل خادماً محلياً:

```bash
npx serve .
```
