# متحف أرض اللبان | Museum of the Land of Frankincense

موقع تعريفي لمتحف أرض اللبان في منتزه البليد الأثري بصلالة — موقع التراث العالمي لليونسكو.

## التشغيل محلياً

افتح `index.html` في المتصفح، أو شغّل خادماً محلياً:

```bash
npx serve .
```

## الصفحات (4)

| صفحة | الملف | المحتوى |
|------|--------|---------|
| الرئيسية | `index.html` | مقدمة، إحصائيات اليونسكو، روابط سريعة |
| المتحف | `museum.html` | عن المتحف، القاعات، المعروضات |
| الزيارة | `visit.html` | ساعات العمل، الخريطة، الفيديو، المعرض، التواصل |
| التذاكر | `tickets.html` | حجز تذاكر الدخول |

## اللغات

العربية، English، Deutsch， Français، **Italiano**

## الملفات

| ملف | الوصف |
|-----|--------|
| `index.html` … `tickets.html` | الصفحات الأربع |
| `style.css` | التصميم والتجاوب للجوال |
| `i18n.js` | الترجمات (خفيف — يُحمّل مرة واحدة ويُخزّن في المتصفح) |
| `script.js` | اللغات، القائمة، المعرض، الحجز |
| `favicon.svg` | أيقونة الموقع |

## الأداء على الجوال

- صفحات منفصلة أخف من موقع واحد طويل
- صور بعرض منخفض + `loading="lazy"`
- فيديو يوتيوب وخرائط Google تُحمّل عند الطلب فقط
- أزرار لغات مختصرة (ع / EN / DE / FR / IT)
